# Ankit Yadav Portfolio

Personal portfolio website for Ankit Yadav, a Frontend and WordPress Developer based in Prayagraj.

## Stack

- React
- JavaScript
- Bootstrap
- Custom CSS

## Highlights

- Westonik experience and live client work
- Project showcase for hospitality, e-commerce, agriculture, FMCG, and agency websites
- Contact section and responsive portfolio layout

## Local Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```

Build output is generated in the `build/` directory.

## Deployment

Recommended deployment target: Cloudflare Pages

- Build command: `npm run build`
- Output directory: `build`
